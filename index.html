Commercial Title: Warble

Academic Title: A System for Multi-Model LLM Evaluation & Optimization

Abstract

Warble is a system designed to automate prompt evaluation and optimisation across multiple Large Language Models (LLMs).

Prompt engineering is often manual and trial-and-error, especially when prompts behave differently across models. Warble addresses this by testing prompts using metrics such as F1-score, ROUGE, and cosine similarity, then optimising them through greedy and evolutionary search.

The system is built using a scalable cloud architecture and provides visual feedback, helping users create more reliable and effective prompts with less manual effort.
